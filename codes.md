Common HTTP Status Codes and Meanings
2xx - Success (Request received and accepted)
200 OK: The request succeeded.
201 Created: Request fulfilled, resulting in a new resource.
204 No Content: Request processed, but no content is returned.
3xx - Redirection (Further action needed)
301 Moved Permanently: Resource has a new permanent URL.
302 Found: Temporary redirect to a new URL.
304 Not Modified: Resource has not changed since last cached.
4xx - Client Error (Request contains bad syntax or cannot be fulfilled)
400 Bad Request: Server cannot process request due to client error.
401 Unauthorized: Authentication is required and has failed or not been provided.
403 Forbidden: Server refuses to authorize the request.
404 Not Found: The requested resource could not be found.
429 Too Many Requests: Rate limiting is in effect.
5xx - Server Error (Server failed to fulfill a valid request)
500 Internal Server Error: Generic error message for unexpected server conditions.
502 Bad Gateway: Server received an invalid response from an upstream server.
503 Service Unavailable: Server is down or overloaded. 
Status Code Categories
1xx Informational: Request received, continuing process.
2xx Success: Action received and understood.
3xx Redirection: Further action needed to complete request.
4xx Client Error: Request contains bad syntax or cannot be fulfilled.
5xx Server Error: Server failed to fulfill a valid request.
